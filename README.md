# INFINITY 👕✨

A INFINITY é um projeto de uma loja de roupas desenvolvido para a disciplina de Experiência do Usuário (UX) da faculdade de Análise e Desenvolvimento de Sistemas.

O objetivo do projeto foi criar uma interface moderna, intuitiva e visualmente atraente, simulando uma plataforma de e-commerce inspirada em grandes lojas de moda online.

Tecnologias Utilizadas
HTML5
CSS3
JavaScript
Funcionalidades
Tela de login
Página inicial com produtos em destaque
Barra de pesquisa
Carrinho de compras visual
Página de perfil
Navegação entre telas
Design responsivo para dispositivos móveis
Observação

⚠️ Este projeto foi desenvolvido exclusivamente para fins acadêmicos e demonstração de conceitos de UX/UI. Algumas funcionalidades são apenas ilustrativas e não realizam operações reais de compra, pagamento ou integração com banco de dados.

Objetivo

Demonstrar conhecimentos em:

Experiência do Usuário (UX)
Interface do Usuário (UI)
Estruturação de páginas web
Design responsivo
Interatividade com JavaScript
